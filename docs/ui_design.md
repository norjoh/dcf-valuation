# UI & UX Design

## Target Users
- Students in valuation & finance clubs
- Junior analysts
- Self-learners in finance & tech

## Core User Flow

1. Select company ticker
2. Verify sector & diagnostics
3. Review historical data
4. Review & edit assumptions
5. Run valuation
6. Inspect sensitivity analysis
7. Inspect comps
8. Download Excel & PPTX

## Key UI Requirements
- Clear warnings & errors
- Interactive sliders for:
  - Growth
  - Margins
  - WACC
  - Terminal growth
- Educational tooltips
- Audit trail panel

## Accessibility
- Easy mode for beginners
- Advanced override mode

## Output Formats
- Excel (.xlsx)
- PowerPoint (.pptx)
- Markdown / PDF summary report
